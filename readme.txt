Przycisk zwrotu zgodny z Dyrektywą UE 2023/2673 — elektroniczne odstąpienie od umowy w WooCommerce.

== Description ==

WC EU Return Button dodaje funkcjonalność elektronicznego odstąpienia od umowy zgodną z Dyrektywą UE 2023/2673.

**Główne funkcje:**
* Przycisk zwrotu w koncie klienta
* Zgodność z dyrektywą UE 2023/2673
* Integracja z WooCommerce
* Panel zarządzania zwrotami

== Installation ==

1. Wgraj folder `wc-eu-return` do katalogu `/wp-content/plugins/`
2. Aktywuj wtyczkę w menu 'Wtyczki' w WordPress
3. Przejdź do WooCommerce → Ustawienia → Zwroty EU

== Frequently Asked Questions ==

= Czy wtyczka jest zgodna z prawem UE? =
Tak, wtyczka implementuje wymogi Dyrektywy UE 2023/2673.

= Czy działa z najnowszym WooCommerce? =
Tak, przetestowana do wersji 9.0.
